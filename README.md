# Freshness-classification-of-Fruits-or-Vegetables-using-YOLOv8s-and-VGG16
Real-time freshness detection of fruits &amp; vegetables using a custom YOLOv8s object detector paired with a VGG16 binary classifier (Fresh/Rotten). Detects 6 categories — Apple, Banana, Carrot, Tomato, Mango, Potato — with confidence-gated classification and low-confidence detections labeled as Others.
